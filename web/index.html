How it works. For every driver we engineer the changing factors that decide a race — starting grid, qualifying pace, recent form, race-pace trend, reliability, constructor strength and weather — and an XGBoost classifier learns the probability of a top-3 finish.

What it learned (in order of impact): grid position → qualifying pace → championship form → constructor strength. Exactly what a race engineer would tell you, learned purely from data.

Data: FastF1 · Model: XGBoost + SHAP · Seasons 2021–2024 · Built as an end-to-end ML project.
